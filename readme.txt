== Description ==

This plug-in is Twitter "@Anywhere" API and "Topsy" API using. 
Visitor's twitter account can be attested putting up connected button to the blog entry. 
After it attests it, link and blog owner's following buttons and blog owner's twitter profile can be displayed. 
Furthermore, the visitor takes side with the page by visitor's account and it is possible to contribute. It is default and there are url on the page and blog owner's twitter account. As a result, my account and blog URL will flow out on twitter, and the conversation extends. 

== Requirements ==

* WordPress 2.9 or later
* PHP 5.0 or later


== Installation ==

1. Unzip the plugin archive and put `tw-anywhere` folder into your plugins directory (`wp-content/plugins/`) of the server. 
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin at "tw anywhere" menu at the WordPress admin panel. Please input API key and username.
1.Twitter API key required "read/write".
